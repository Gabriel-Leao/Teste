HTML é igual a significado, ou seja semântica.

CSS é igual a estilo, ou seja forma.

Principais Formatações

Negrito / Destaque

Sempre que quiser dar destaque ao texto e deixar em negrito use a tag <strong>. Assim o termo em destaque tem semântica ou seja significado

Itálico / Ênfase

Para deixar o termo em itálico ou em ênfase use a tag <em>. Para que o termo em destque seja semãntico e tenha significado

Texto Marcado

Podemos criar um texto marcador com a tag <mark>

Essa tag marca um trecho do texto igual a um marcador comum. Sua cor padrão é amarelo, mas podemos muda-lá através do CSS ou a tag style.

Texto grande e Pequeno

A tag big se tornou obsoleta, mas a tag <small> ainda é válida e deixa o texto pequeno. Para criar um texto grande utilize a formatação do CSS.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas desconsiderado

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adiconado depois.
